Instructions

1. Connect GitHub with Device Login.

2. Pick an existing repository or create a new private one.

3. Confirm the branch, sync mode, and ignore recommendations.

4. Run Sync Now, Clean Upload, or Cancel Current Upload as needed.

GitHub Device Login

Private repositories are supported. Make sure the GitHub token has access to the selected private repo and the branch name is correct.
