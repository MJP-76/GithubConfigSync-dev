Instructions

1. Connect GitHub with Device Login.

2. Pick an existing repository, or create `ha-github-config-sync` (private by default, public optional).

3. Check the folder structure inclusions/exclusions and the .gitignore files.

4. Run Sync Now, Clean Upload, or Cancel Current Upload as needed.
